# deepstream_8_0
Este projeto fornece um ambiente Docker minimalista para execução do NVIDIA DeepStream SDK 8.0, utilizando aceleração por GPU NVIDIA. A solução foi pensada para rodar de forma estável em sistemas Linux modernos (ex.: Ubuntu 24.04 no host), isolando todas as dependências críticas — CUDA, TensorRT, GStreamer e bibliotecas NVIDIA.
